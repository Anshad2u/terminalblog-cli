# terminalblog CLI 📟

> Fetch latest AI coding agent articles straight from your terminal.

```bash
npx terminalblog
```

## Usage

```bash
npx terminalblog          # Show latest articles
npx terminalblog ls       # Same as above
npx terminalblog compare  # Compare coding agents side by side
npx terminalblog help     # Show help
```
